使用HEXO在GITHUB PAGES上建立博客

在Github上注册账号

在Github网站上免费注册一个账号即可。

创建仓库

新建一个repository, 注意这个repository的名字要和注册的用户名对应, 如我的用户名是WenzelHu, 那么repository的名字应该为wenzelhu.github.io。 这个repository的分支就设为默认的master。 这样就可以通过链接https://wenzelhu.github.io/访问这个页面了，虽然现在这个页面上没有什么内容。

安装Hexo

这个时候就需要使用Hexo来编辑博客页面的内容。Hexo是一个博客框架，使用它我们可以很方便地编辑博客内容并把内容部署到页面上去。要使用Hexo，需要系统里有Nodejs和Git。下面来一步步安装好Hexo。

  • 安装Nodejs
    下载并安装Nodejs
    关于Nodejs，参考此网站
  • 安装Git
    下载并安装Git
  • 安装Hexo
    在命令提示符里运行下面的命令

常用的几个hexo命令

  • hexo generate (hexo g) 生成静态文件，在当前的blog文件夹内生成一个public文件夹
  • hexo server (hexo s) 启动本地web服务，用于预览博客
  • hexo new “your file name”(hexo n “your file name”) 生成新的Post文章
  • hexo deploy (hexo d) 把静态文件部署到远端 (github)
  • hexo d -g 生成后部署
  • hexo s -g 生成后预览

此时打开 http://localhost:4000/，可以看到hexo内置的Hello World博文。
